Installation

We have used Laravel Framework to craft best possible support center :

  • Database
    Change database settings in app/config/database.php . Change database name , username and password in mysql section and save the file .
  • Upload Files to Server
    Upload all files from ESHOP folder to your server . SSH to your server and run below commands

    php composer install which will install all dependencies

    php artisan migrate which will create all tables schema

    php artisan db:seed which will insert our demo admin

    By default admin email is admin@mail.com and password is admin
  • Permissions
    You need to give write permission to app/storage directory . Other directories/files which require write access are app/views/emails , public/uploads , app/config/mail.php and app/config/packages/bogardo/mailgun/config.php

    Example

    chgrp -R www-data /your-uploaded-folder

    chmod -R 775 /your-uploaded-folder/storage

    chmod -R 775 /your-uploaded-folder/public/uploads

Setting up Mail

Go to config folder you will find a file named mail.php in that configure your mailgun settings.

Stripe

Go to config folder you will find a file named services.php in that configure your stripe settings.

Also go ESHOP\resources\views\frontend\cart.blade.php on line no 91 set live key of stripe.
